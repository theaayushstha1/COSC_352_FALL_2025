# Project 7 – Baltimore Homicide Dashboard (2021–2025)

This project uses **R + RShiny** to scrape, clean, and visualize Baltimore homicide data from Cham's Page for years 2021–2025.

## Features
- Web scraping with rvest
- Data cleaning with janitor & dplyr
- Interactive dashboard with:
  - Year filter
  - Date range filter
  - Age range filter
  - Text search
- Plots:
  - Homicides by year
  - Homicides by month
- Full data table (DT)
- Fully Dockerized
