# Project 8: Baltimore Homicide Functional Analysis

Pure functional programming in Clojure analyzing Baltimore homicide data.

## Analyses
1. **Temporal Trends**: Homicides by year/month with YoY changes
2. **Neighborhood Hot Spots**: Top 10 neighborhoods with clearance rates

## Usage
```bash
# Generate sample data
chmod +x generate_sample_data.sh
./generate_sample_data.sh

# Run analysis
chmod +x run.sh
./run.sh
```

## Features
- Pure functional programming
- Immutable data structures
- Higher-order functions
- Function composition
